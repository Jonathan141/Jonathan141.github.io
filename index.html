Blog发布流程（使用Hexo）

第一：创建要发布的blog

手动创建新文件

或使用命令行创建新文件

第二：启动Hexo本地服务器，并开启调试模式

默认访问http://localhost:4000,检查站点是否正常运行
预览的同时可以修改文章内容或主题代码，保存后刷新页面即可

第三：部署到网站

hexo generate或hexo g
生成网站静态文件到默认设置的public文件夹
hexo deploy或hexo d
自动生成网站静态文件，并部署到github设定的仓库

Hexo 是基于 Node.js 开发的一个静态博客生成器，提供本地实时预览及部署功能。

使用前确保电脑中以安装 Node.js Git

基本使用方式如下：

安装完成后，在指定文件夹中新建所需的文件

注意：配置项与配置内容之间有一个空格，不可省略

第二：安装自动发布插件

第三：在相应文件夹内新建md文件

手动创建新文件

或使用命令行创建新文件
