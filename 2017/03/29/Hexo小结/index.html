Hexo使用小结

电脑环境配置

Hexo 是基于 Node.js 开发的一个静态博客生成器，提供本地实时预览及部署功能。

使用前确保电脑中以安装 Node.js Git

基本使用方式如下：

安装完成后，在指定文件夹中新建所需的文件

自动发布 Hexo 搭建的博客

第一：先配置 _config.yml 文件
注意：配置项与配置内容之间有一个空格，不可省略

第二：安装自动发布插件

第三：在相应文件夹内新建md文件
手动创建新文件
source => _posts => xxx.md
或使用命令行创建新文件

新建一篇文章，如果没有设置 layout 的话，默认使用 _config.yml 中的 default_layout 参数代替。如果标题包含空格的话，请使用引号括起来。
第四：在博客文件夹根目录下使用命令一键发布

发布可能有延时，稍加等待即可
